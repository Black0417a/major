# MajorTI

MajorTI 是一个面向高考考生的匿名志愿兴趣测评系统原型。它包含问卷测评、兴趣画像、专业推荐、专业筛选、详情和对比功能。

## 运行

```bash
npm run seed
npm start
```

打开 `http://localhost:5173`。

## 当前实现

- 无需登录的匿名测评流程，作答进度和结果不落库、不写 localStorage
- 5 大维度问卷：学科倾向、职业兴趣、能力特长、性格特质、价值观
- 700+ 本科专业种子数据，当前只展示有来源依据的专业名称、所属门类和专业类
- 专业向量匹配推荐算法
- 专业详情、筛选和最多 4 个专业对比
- 推荐结果即时生成，刷新页面后测评状态清空，避免多人部署时数据串扰

## 数据来源规划

生产环境应将 `data/majors.json` 替换为经过逐条校验的权威数据集，来源包括教育部本科专业目录、阳光高考专业库、高校培养方案、各省教育考试院录取分数线、国家统计局行业工资数据、高校就业质量报告与权威行业报告。当前原型不展示未接入权威来源的课程、就业方向、就业率、薪资、录取分数线等数据。

## 隐私与并发

当前运行时不保存用户进度：

- 前端不使用 `localStorage`、`sessionStorage` 或 Cookie 保存答案。
- 后端不保存 session、答案快照或测评结果。
- `/major/api/assessment/submit` 只在单次请求内计算推荐结果并立即返回。
