# MajorTI

MajorTI 是一个面向高中生的兴趣型专业推荐原型。

## 现在支持

- 纯兴趣测评，不需要填写地域、分数或学校信息
- 更适合“还不太确定自己想做什么”的题目表达
- 测评过程带加载动画和骨架屏
- 可选邮箱发送测评结果，留空也能继续使用
- 填写邮箱后会通过 SMTP 发送结果到邮箱
- 专业库支持搜索、筛选、详情和对比
- 专业名称、门类、专业类对齐官方目录数据，页面只展示已整理的目录级信息

## 运行

```bash
npm start
```

打开控制台打印的本地地址，默认从 `5173` 起，若被占用会自动换到下一个可用端口。

## 数据说明

- `data/majors.json` 用作专业目录数据源
- 后端只在用户输入邮箱后发送测评结果，并保存一份记录
- 当前推荐逻辑只基于兴趣向量，不使用地域、分数等信息
